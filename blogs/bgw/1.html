    vkuyxu 2023/1/1 HTML前端面试题

    # 1.src 和 href 的区别

    src 用于替换当前元素，href 用于在当前文档和引用资源之间确立联系。

    src 指向外部资源的位置，指向的内容将会嵌入到文档中当前标签位置。会将 src 资源下载并应用到文档内。

    href 指向网络资源所在位置，建立和当前文档之间的链接。

    # 3.script 标签中 defer 和 async 的区别。

    defer 和 async 属性都是去异步加载外部 js 脚本文件，它们都不会阻塞页面的解析。

    defer 会按照他们在页面中的顺序进行加载，与页面并行加载，等待页面加载完毕后在执行。

    async 也是与页面并行加载，但是在脚本加载完毕后就会立刻执行，如果此时页面还没有加载完毕，那

    么就会阻塞页面的渲染。

    # 4.常用的 meta 标签有哪些？

    meta 标签有 name 和 content 属性定义，用来描述网页文档的属性。

    charset ： 用来描述 HTML 文档的编码类型。

    keywords:页面关键词。

    description:页面描述。

    refresh:页面重定向和刷新。

    viewprot，适配移动端，可以控制视口的大小和比例。

    conent 参数 ：

    1. width viewport 宽度

    2. height viewport 高度

    3. initial-scale 初始缩放比例 4. maximum-sacle 最大缩放比例

    4. minmum-sacle 最小缩放比例

    5. user-scalable 是否允许用户缩放

    搜索引擎索引方式:

    name = "robots"

    content 参数：

    1. all：文件被检索，链接可以被查询。

    2. none:文件不被检索，链接不可被查询。

    3. index：文件被检索。

    4. follow:页面上链接可以被查询。

    5. noindex：文件不被检索。

    6. nofollow:链接不可以被查询。

    # 5.HTML5 有哪些更新?

    1. 新增语义化标签:nav，header footer aside section article

    2. 音频，视频标签: audio ,video

    3. 数据存储：localStorage，sessionstorage

    4. canvas 画布 Geolocation 地理定位 websocket 通信协议

    5. input 新增属性:placeholder ，autocomplete，aotofocus，required

    6. hsitory API : go，forward，back，pushstate

    # 6.Img 的 srcset 属性的作用？

    提供多个备选的图片源，以便可以根据设备的大小和分辨率来选择合适的图片进行加载，从而实现响

    应式的图像选择和高分辨率图像提供

    <img
    src="image-128.png"
    srcset="image-128.png 128w, image-256.png 256w, image-512.png 512w"
    sizes="(max-width: 360px) 340px, 128px" />

    其中 srcset 指定图片的地址和对应的图片质量。sizes 用来设置图片的尺寸零界点。对于 srcset 中的 w 单位，可以理解成图片质量。如果可视区域小于这个质量的值，就可以使用。浏览器会自动选择一个最小的可用图片。

    # 7.行内元素有哪些?块级元素有哪些？空元素有哪些?

    1. 行内元素有：a,b,span,img,input,select,strong

    2. 块级元素：div ol ul dl dt dd h1~h6

    3. 空元素:没有闭合标签 br hr img input link meta

    # 8.iframe 有哪些优点和缺点？

    iframe 元素会创建包含另外一个文档的内联框架（行内框架）

    优点:

    用来加载速度较慢的内容（如广告）

    可以使脚本并行下载

    可以实现跨子域通信

    缺点:

    会阻塞主页面的 onload 事件。

    无法被一些搜索引擎搜索识别

    会产生很多页面不容易管理。

    # 9.Canvas 和 SVG 的区别。

    canvas :基于像素的绘图方式，依赖于分辨率。图形在缩放时可能会失真或产生锯齿状的边缘。

    svg:基于矢量的绘图方式，使用 XML 格式来描述图形。图形可以无损地缩放和旋转，并且保持较好的质量。

    # 10.渐进增强和优雅降级的区别。

    渐进增强 主要是针对低版本的浏览器进行页面重构，保证基本功能的情况下，在针对高级的浏览器进行效果，交互等方面的改进和追加功能，以达到更好的用户体验。

    优雅降级 就是一开始就构建完整功能，然后在针对低版本的浏览器做兼容。
